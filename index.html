Executive Summary

This project implements a secure, segmented enterprise network environment. The architecture separates critical infrastructure (LAN1) from client workstations (LAN2), managed by a pfSense firewall. Key security features include centralized Active Directory authentication, Suricata IDS, Wazuh SIEM monitoring, and strict firewall policies.

Project Conclusion

This project successfully demonstrates a secure, segmented enterprise network. All deliverables have been met, including:

  • Full segmentation between Server (LAN1) and Client (LAN2).
  • Centralized authentication via Active Directory.
  • Real-time security monitoring with Wazuh SIEM & Suricata.
  • Automated policy enforcement via GPO.

Lessons Learned

Key takeaways include the importance of precise firewall rule ordering, the necessity of FIM for tracking unauthorized changes, and the effectiveness of centralized logging for correlating security events across different operating systems.
